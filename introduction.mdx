DAAAM (Describe Anything, Anywhere, at Any Moment) is a CVPR 2026 research system from MIT-SPARK that gives robots real-time spatio-temporal scene understanding. It feeds every RGB frame through SAM segmentation, BotSort temporal tracking, and DAM semantic grounding, then integrates the results into Hydra to produce hierarchical 4D Dynamic Scene Graphs — capturing what objects are, where they are in 3D space, and when they were seen, all at real-time framerates.

## Key contributions

DAAAM introduces three core advances over prior scene graph systems:

- **Novel optimization-based frontend** — a frame-selection and assignment pipeline that minimizes the number of frames queried while maximizing object saliency, using CVXPY mixed-integer programming under the hood.
- **Hierarchical 4D scene graph construction** — objects, places, and rooms organized in a multi-layer graph with full spatio-temporal history, built on the Hydra DSG framework.
- **State-of-the-art benchmark results** — top performance on the NaVQA and SG3D benchmarks for scene-graph-based navigation and question answering.
